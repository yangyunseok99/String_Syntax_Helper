커스텀 태그 규칙 관리

사용 방법 안내

0

API 키 설정 (최초 1회 필수)

번역 기능을 사용하려면 무료 AI API 키가 필요합니다.
1. Google AI Studio에 접속하여 API Key를 생성합니다.
2. 우측 상단의 API 키 설정 버튼을 누르고 발급받은 키를 붙여넣습니다.
입력된 키는 서버로 전송되지 않으며, 현재 사용 중인 브라우저(내 PC)에만 안전하게 저장됩니다.

1

텍스트 태그 입히기

[원본] 에디터에서 텍스트를 마우스로 드래그하여 선택한 후, 상단의 색상 적용이나 퀵 태그 버튼을 클릭하세요. 선택한 영역에 즉시 HTML 태그가 적용됩니다.

2

태그 유지 자동 번역

태그 작업이 완료되면 중앙의 드롭다운에서 목표 언어를 선택하고 번역 실행 버튼을 누릅니다.
* 태그 구조(색상, 강조 등)는 100% 유지된 채 텍스트 내용만 번역됩니다!

3

커스텀 태그 규칙

팀 내 규약에 맞는 특수한 태그(예: <color name="red">)가 있다면, 하단 패널에서 규칙을 추가해 보세요. 버튼 이름과 시작/종료 태그를 입력하면 즉시 사용할 수 있는 나만의 버튼이 생성됩니다.

4

미리보기 및 복사

우측의 실시간 렌더링 미리보기에서 결과물을 시각적으로 확인하세요. 작업이 끝나면 에디터 하단의 복사 버튼을 클릭하여 클립보드에 바로 가져갈 수 있습니다.
