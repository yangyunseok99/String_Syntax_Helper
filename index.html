색상 지정:
퀵 태그
실시간 렌더링 미리보기
LIVE PREVIEW

커스텀 태그 규칙 관리

추가한 규칙은 브라우저에 자동 저장됩니다

상세 사용 가이드

1

텍스트 입력 및 기본 태그 적용

스트링 에디터에 텍스트를 입력합니다. 태그를 입히고 싶은 부분을 마우스로 드래그하여 선택한 후, 툴바에 있는 퀵 태그 버튼(강조, 기울임 등)을 클릭하면 즉시 태그가 적용됩니다.

2

원하는 색상으로 지정

글자 색상을 바꾸고 싶다면, 툴바 좌측의 색상 팔레트를 클릭하거나 HEX 코드(예: #FF0000)를 직접 입력하세요. 텍스트를 드래그하고 색상 적용 버튼을 누르면 해당 색상의 스타일 태그가 씌워집니다.

3

잘못 입힌 태그 제거하기

실수로 태그를 잘못 적용했거나 순수 텍스트만 남기고 싶을 때는, 해당 영역을 드래그하고 우측 상단의 태그 제거 버튼을 눌러주세요. 내용물은 유지되고 태그만 깔끔하게 지워집니다.

4

나만의 커스텀 규칙 저장

팀 프로젝트마다 자주 쓰는 태그(예: <color name="red">)가 있다면, 하단 패널에 버튼 이름, 시작 태그, 종료 태그를 입력하고 규칙을 추가하세요. 추가된 규칙은 브라우저에 영구적으로 자동 저장됩니다.

5

실시간 확인 및 결과물 내보내기

우측의 LIVE PREVIEW 화면에서 텍스트가 어떻게 렌더링되는지 실시간으로 확인하세요. 작업이 완료되면 에디터 하단의 결과물 복사 버튼을 눌러 클립보드에 담아갈 수 있습니다.

처리되었습니다!
